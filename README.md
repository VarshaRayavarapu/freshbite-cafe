# FreshBite Café

FreshBite Café is a healthy juice and sandwich ordering website.

## Features
- Responsive HTML website
- CSS styling
- JavaScript-based order customization
- Git version control
- GitHub repository hosting
- GitHub Actions CI/CD pipeline
- Automated deployment with GitHub Pages

## Tools Used
- VS Code
- HTML
- CSS
- JavaScript
- Git
- GitHub
- GitHub Actions
- GitHub Pages

## DevOps Workflow
1. Develop code in VS Code
2. Track changes using Git
3. Push code to GitHub
4. Trigger GitHub Actions pipeline
5. Automatically deploy to GitHub Pages
